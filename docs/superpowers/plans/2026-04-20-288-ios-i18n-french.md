# iOS Internationalization Layer (French) — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Ship a complete iOS i18n layer using Apple String Catalogs so that the app renders in the user's system language (English source + French, with English fallback for any other locale) and contains zero hardcoded user-facing strings.

**Architecture:** Single `Localizable.xcstrings` String Catalog at `apps/ios/Pebbles/Resources/`. Three call-site patterns: (A) SwiftUI literals auto-extracted on build, (B) Swift-level static copy typed as `LocalizedStringResource`, (C) DB reference names (`Emotion`, `Domain`, `EmotionRef`, `DomainRef`) resolved via slug-keyed catalog entries with the DB `name` column as a last-resort fallback. Language resolution is 100% system-driven via `CFBundleLocalizations` — no in-app picker.

**Tech Stack:** SwiftUI, iOS 17+, Swift 5.9, Xcode 15+, Apple String Catalog (`.xcstrings`), Swift Testing. No new third-party dependency.

**Spec:** `docs/superpowers/specs/2026-04-20-ios-i18n-french-design.md`

**Issue:** #288. Branch `feat/288-ios-i18n-french` already checked out.

```swift
import Foundation
import Testing
@testable import Pebbles

@Suite("Localization — Pattern C fallback")
struct LocalizationPatternCTests {

    @Test("Emotion.localizedName falls back to name when slug has no catalog entry")
    func emotionFallsBackToName() {
        let emotion = Emotion(
            id: UUID(),
            slug: "not-a-real-slug-xyz",
            name: "Fallback Name",
            color: "#000000"
        )
        #expect(emotion.localizedName == "Fallback Name")
    }

    @Test("Domain.localizedName falls back to name when slug has no catalog entry")
    func domainFallsBackToName() {
        let domain = Domain(
            id: UUID(),
            slug: "not-a-real-slug-xyz",
            name: "Fallback Name",
            label: "ignored label"
        )
        #expect(domain.localizedName == "Fallback Name")
    }

    @Test("EmotionRef.localizedName falls back to name when slug has no catalog entry")
    func emotionRefFallsBackToName() {
        let ref = decodeRef(EmotionRef.self, json: """
        {"id":"00000000-0000-0000-0000-000000000000","slug":"not-a-real-slug-xyz","name":"Ref Fallback","color":"#000000"}
        """)
        #expect(ref.localizedName == "Ref Fallback")
    }

    @Test("DomainRef.localizedName falls back to name when slug has no catalog entry")
    func domainRefFallsBackToName() {
        let ref = decodeRef(DomainRef.self, json: """
        {"id":"00000000-0000-0000-0000-000000000000","slug":"not-a-real-slug-xyz","name":"Ref Fallback"}
        """)
        #expect(ref.localizedName == "Ref Fallback")
    }

    // MARK: helpers

    private func decodeRef<T: Decodable>(_ type: T.Type, json: String) -> T {
        // swiftlint:disable:next force_try
        return try! JSONDecoder().decode(T.self, from: Data(json.utf8))
    }
}
```

```swift
/// Resolve `LocalizedStringResource` against a specific locale for testing.
private func resolve(
    _ resource: LocalizedStringResource,
    locale: Locale
) -> String {
    var copy = resource
    copy.locale = locale
    return String(localized: copy)

/// Resolve a runtime-key String (Pattern C) against a specific locale.
private func resolveKey(
    _ key: String,
    locale: Locale,
    fallback: String
) -> String {
    let resource = LocalizedStringResource(
        String.LocalizationValue(key),
        defaultValue: String.LocalizationValue(fallback),
        locale: locale
    )
    return String(localized: resource)
}
```
